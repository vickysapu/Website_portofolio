Data Tables

Material Admin Pro utilizes Simple DataTables as the foundation for the interactive data tables in this toolkit. This third-party plugin is a lightweight, dependency-free, JavaScript only table plugin, perfect for creating filterable, searchable data tables inside an application.

Example Data Table

The Simple DataTable example below features sortable columns, search, pagination, and number of entries selection. The plugin has been restyled within this toolkit to match Material Design's table styling. For instructions on how to use these data tables in an app, visit the data tables plugin documentation page.
